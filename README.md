# vue-start
## 1.项目描述
这是一个vue的种子项目，项目结构参考react+redux的项目目录，可作为作为一般项目的种子

技术栈如下：
1. vue
2. vue-router
3. vuex
4. axios
5. vue-axios
6. node-sass
7. webpack
8. es6
9. express
10. mockjs
11. mocha
12. superagent
13. html5shiv

## 2.项目使用
### 克隆到本地
`git clone https://github.com/shifeng1993/vue-start.git your-app-name`
### 进入文件夹并安装依赖
`cd your-app-name`

`npm install`

### 运行开发环境
`npm run dev`

### 打包编译
`npm run build`

### 单元测试
`npm run unit`

### 自动化测试
`npm run e2e`

### 全部测试
`npm test`

目前项目使用的vue版本为2.2.2，vuex版本为2.2.1，axios版本为0.15.3，webpack版本为2.2.1，最低node版本要求为4.0.0以上
