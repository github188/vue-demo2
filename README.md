# 如何快速开始一个工程项目
## 1.项目描述
注：本项目使用的脚手架均为 https://github.com/shifeng1993/vue-start
技术栈如下：
1. vue
2. vue-router
3. vuex
4. axios
5. vue-axios
6. node-sass
7. webpack
8. es6
9. express
10. mockjs
11. mocha
12. superagent
13. html5shiv

## 2.项目使用
### 克隆到本地
`git clone https://github.com/shifeng1993/demo1.git`
### 进入文件夹并安装依赖
`cd demo1`

`npm install`

### 运行开发环境
`npm run dev`

### 打包编译
`npm run build`

目前项目使用的vue版本为2.2.2，vuex版本为2.2.1，axios版本为0.15.3，webpack版本为2.2.1，最低node版本要求为4.0.0以上
